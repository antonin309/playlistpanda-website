# Playlist Panda

Website for my Spotify playlist curation brand. No framework, just clean HTML and CSS. Built to load fast and look good.

## Pages

- **Home** (`index.html`) — hero, featured playlists, curator pitch
- **Blog** (`blog.html`) — articles on playlist curation and music marketing
- **Musicology** (`musicology.html`) — deep-dives into music history (DDR hip-hop, etc.)
- **Course** (`kurs.html`) — paid course landing page for aspiring curators
- **Legal** (`datenschutz.html`, `impressum.html`) — GDPR compliance pages

## Tech stack

- Vanilla HTML · CSS custom properties · JavaScript
- Google Fonts (Instrument Sans, DM Mono)
- No build step — open any `.html` file directly

## Design tokens

```css
--bg:      #faf7f2   /* warm off-white */
--accent:  #e07060   /* coral/salmon */
--teal:    #3bbfb8
```

## Getting started

```bash
# No install needed — just open in browser
open index.html

# Or serve locally
npx serve .
```
